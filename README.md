# Scripts

A collection of short utility scripts for common development tasks, organised by platform.

| Folder | Platform |
|--------|----------|
| [`mac/`](mac/README.md) | macOS — bash scripts |
| [`linux/`](linux/README.md) | Linux — bash scripts |
| [`windows/`](windows/README.md) | Windows — batch (`.bat`) scripts |

## Categories

- **Git shortcuts** — `gacp`, `gc`, `gc.` / `gcd`, `gcfd`, `gp`, `gpuo`, `grh`, `grho`, `gs`
- **npm shortcuts** — `nb`, `nsd`, `nt`
- **Workspace launchers** — `itabs`, `mm`

See the README inside each platform folder for setup instructions and a full script reference.
